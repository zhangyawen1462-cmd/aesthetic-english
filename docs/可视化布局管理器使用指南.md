# 🎨 可视化布局管理器使用指南

## 🎯 这是什么？

一个超级直观的拖拽工具，让你完全掌控每个页面显示什么内容。

**不需要记住复杂的 Display_Position 和 Sort_Order！**

只需要：
1. 打开布局管理器
2. 拖拽封面到对应位置
3. 实时预览效果
4. 保存布局

---

## 📍 访问地址

```
http://localhost:8080/admin/layout-manager
```

或者从管理后台首页进入：
```
http://localhost:8080/admin
```

## 🚀 使用步骤

### 第一步：选择页面

点击顶部标签页，选择要管理的页面：

```
┌─────────────────────────────────────────┐
│ [🏠 Dashboard] [🎬 Daily Cinema]        │
│ [🧠 Cognitive] [💼 Business]            │
└─────────────────────────────────────────┘
```

### 第二步：拖拽封面

从下方的"可用课程"列表，**用鼠标拖动封面图片**到上方的布局区：

```
┌─ 布局区域 ─────────────────────────┐
│ [空位 1] [空位 2] [空位 3]         │
│ ↑ 拖拽封面到这里                   │
└────────────────────────────────────┘

📚 可用课程：
┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐
│封面1│ │封面2│ │封面3│ │封面4│
│图片 │ │图片 │ │图片 │ │图片 │
└─────┘ └─────┘ └─────┘ └─────┘
  ↑ 鼠标按住封面图片，拖到上面
```

**操作方法**：
1. 鼠标移到封面图片上
2. 按住鼠标左键不放
3. 拖动到上方的布局区
4. 松开鼠标，封面就添加进去了

### 第三步：调整顺序

在布局区内**拖动封面图片**，调整显示顺序：

```
[封面1] [封面2] [封面3]
   ↓ 按住封面拖动
[封面2] [封面1] [封面3]
```

**操作方法**：
1. 鼠标移到布局区的封面上
2. 按住鼠标左键
3. 拖到想要的位置
4. 松开鼠标

### 第四步：预览效果

点击右上角的"预览"按钮，在新标签页查看实际效果。

### 第五步：保存布局

满意后，点击右下角的"💾 保存布局"按钮。

## 🎯 使用场景

### 场景 1：更新 Dashboard 精选

**需求**：想把最新的 3 期课程放到 Dashboard

**操作**：
1. 打开布局管理器
2. 切换到"Dashboard 精选"标签
3. 点击"清空"按钮（如果有旧内容）
4. 从下方拖拽 3 个最新课程到布局区
5. 调整顺序
6. 保存

**结果**：Dashboard 立即显示新的 3 期课程

---

### 场景 2：精选 Daily Cinema 视频

**需求**：从 20 期 Daily 视频中，精选你最喜欢的几期

**操作**：
1. 切换到"Daily Cinema"标签
2. 浏览下方的所有 Daily 视频
3. 拖拽你认为最好的视频到布局区（3-6 期都可以）
4. 按你想要的顺序排列
5. 保存

**结果**：Daily Cinema 只显示你精选的视频

---

### 场景 3：更换 Cognitive 精选

**需求**：把旧的 Cognitive 课程换成新的

**操作**：
1. 切换到"Cognitive 精选"标签
2. 点击旧课程右上角的"✕"按钮移除
3. 从下方拖拽新课程到布局区
4. 保存

**结果**：Cognitive 精选区显示新课程

## 💡 操作技巧

### 1. 拖动封面的正确姿势

**桌面端（鼠标）**：
1. 鼠标移到封面图片上
2. 按住鼠标左键不放
3. 移动鼠标到目标位置
4. 松开鼠标左键

**移动端（触摸）**：
1. 手指按住封面图片
2. 拖动到目标位置
3. 松开手指

### 2. 快速清空

点击右上角的"清空"按钮，一键移除所有课程。

### 2. 快速清空

拖拽过程中，布局区会实时显示效果，所见即所得。

### 3. 实时预览

鼠标悬停在课程上，点击右上角的"✕"按钮。

### 4. 移除单个课程

每个课程左上角显示序号（#1、#2、#3...），表示显示顺序。

### 5. 查看序号

- Dashboard：只显示 Daily 分类的课程
- Daily Cinema：只显示有视频的 Daily 课程
- Cognitive：只显示 Cognitive 分类的课程
- Business：只显示 Business 分类的课程

## ❓ 常见问题

### Q1: 为什么有些课程不显示在可用列表中？

**A:** 可能的原因：
1. 课程已经添加到布局中
2. 课程的分类不匹配（例如在 Dashboard 标签页只显示 Daily 分类）
3. 课程的 Status 不是 Published

### Q2: 保存后多久生效？

**A:** 立即生效！保存后刷新对应页面即可看到更新。

### Q3: 可以同时显示在多个位置吗？

**A:** 可以！同一个课程可以同时添加到 Dashboard 和 Daily Cinema。

### Q4: 拖拽不工作怎么办？

**A:** 
1. 确保浏览器支持拖拽（推荐使用 Chrome）
2. 刷新页面重试
3. 检查浏览器控制台是否有错误

### Q5: 如何恢复默认布局？

**A:** 点击"清空"按钮，然后重新拖拽课程。

| **出错概率** | 容易填错 | 几乎不会出错 |
